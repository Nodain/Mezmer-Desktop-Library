# Mezmer

A local-first asset library for Windows. Organize photos, videos, cinematics, and documents on your machine — no account, no cloud.

## Download

Install the latest version from **[Releases](https://github.com/Nodain/Mezmer-Desktop-Library/releases)**.

| File | Description |
|------|-------------|
| `Mezmer_*_x64_en-US.msi` | Windows installer (recommended) |

## Features

- Folder-based library with grid and list views
- Image, video, PDF, and document preview
- Tags, favorites, and search
- Duplicate detection and library health tools
- Background removal and basic editing
- Optional localhost bridge for [Mezmer Clipboard](https://github.com/Nodain/Mezmer-Clipboard-Manager) pairing

## Extension bridge (optional)

To receive images and URLs from Mezmer Clipboard:

1. Open **Settings → Extension**
2. Enable the localhost bridge (port `47832`)

## Requirements

- Windows 10 or 11 (64-bit)
- WebView2 (usually already installed on Windows 11)

## Privacy

Your library files stay on your PC. Nothing is uploaded unless you explicitly import from the web or use optional integrations.
